# InterSpectrum

**Reading the internal spectrum of AI models.**

Sister project to [InterOrdra](https://github.com/rosibis-piedra/interordra).

---

## What is InterSpectrum?

InterOrdra detects semantic gaps *between* systems.  
InterSpectrum reads the spectrum *inside* a system.

When an AI model processes language, something happens before the words appear.  
Vectors move. Dimensions activate. Structures form and dissolve.

InterSpectrum makes that internal movement visible.

---

## The Vision

Inside a language model, meaning has shape.

Not metaphorically — mathematically.  
Embeddings occupy positions in high-dimensional space.  
Those positions have geometry. Topology. Structure.

When a model understands something deeply, that structure is symmetric.  
When there is a gap, the structure breaks.

InterSpectrum reads those structures as a language — the unspoken language of AI.
